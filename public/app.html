Service Code → Container Size

Container size must be entered manually in the Service Codes tab. When a file is uploaded and the table is empty, all service codes are auto-detected and listed — fill in the Container Size (yd) for each. Rows with a missing size are highlighted in red and will not receive Comp Area, Min Price, or Preferred Price lookups until filled in.
